# Canvas Setup Guide — MGMT162 Spring 2026

This document tells you how to set up your Canvas course using the generated pages.

---

## Generated Pages

### Session Pages (one per class)

Located in `Sessions/`. 3 session pages generated.

### Week Overview Pages

Located in `Weeks/`. 2 week pages generated.

### Course Schedule

Located at `Course Schedule.md`. Master schedule table.

---

## How to Add Pages to Canvas

1. Create one module per week (Week 1, Week 2, ...)

2. In each module, add the week overview page first, then the two session pages

3. In the Canvas page editor, paste the markdown content (or convert to HTML)

4. **Publish week by week** — keep future weeks unpublished
