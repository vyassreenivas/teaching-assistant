**Priority tiers:**

- HIGH (behavioral, build through repetition): Q1.4 Enthusiasm, Q1.5 Interaction, Q1.6 Respect
- MONITOR (systemic, harder to move): Q1.1 Overall, Q1.8 Workload, Q1.9 Learning

---

## The 5 Mantras

1. **Grade everything within 7 days.** This is the single highest-leverage SET move. Q1.7 (Feedback) is the most controllable critical question. Late grading erodes trust faster than anything else.

2. **Display an agenda every class.** Q1.2 (Organization) is visible and cheap. A clear agenda slide at the start signals that class time is structured and respected.

3. **Say "my door is open" aloud every 2 classes.** Q1.5 (Interaction) is built by repetition, not grand gestures. Students need to hear availability mentioned regularly, not just see it on the syllabus.

4. **Pulse survey at Classes 7-8 and 14-15, close the loop next class.** "You said X, here's what I changed" is the most powerful SET move after grading. Students need to see their feedback acted on.

5. **Protect respect (Q1.6).** This is the superpower. Never compromise it — not for humor, not for efficiency, not for anything. Respect is the foundation that makes everything else work.

## Pre-class Checklist

Run before EVERY class. No exceptions.

- [ ] Agenda slide ready and will be displayed (Q1.2)
- [ ] Key takeaways identified for recap at end (Q1.2)
- [ ] Activities and materials prepped (Q1.3)
- [ ] Grading status: anything due within 7 days posted or on track (Q1.7)
- [ ] Plan to mention office hours / availability? (Q1.5 — every 2 classes)
- [ ] Quiz printed + Canvas version live, if applicable
- [ ] Slides reviewed for student names (FERPA)
- [ ] Class log template ready

## Post-class Checklist

Run after EVERY class. Same day, before moving on.

- [ ] Raw notes dumped into class log
- [ ] Class log swept for student names → replaced with IDs
- [ ] Tracker updated: class status, new action items
- [ ] Skill log updated if something notable happened
- [ ] Look-ahead: is the NEXT class fully prepped?

## Quarter Roadmap (4 phases)

| Phase | Classes | Focus |
|-------|---------|-------|
| Foundation | 1–4 | Set expectations, establish rhythm, learn names |
| Build + Survey | 5–8 | Deepen rigor, first pulse survey, close the loop |
| Respond + Improve | 9–14 | Act on feedback, midterm, maintain momentum |
| Finish Strong | 15–20 | Final projects, second survey, strong close |

---

## Weekly SET Check-in (~10 min, Sundays)

1. Review grading queue: anything overdue or approaching 7-day limit?
2. Count classes since last "door is open" mention — due if ≥2.
3. Check if pulse survey window is approaching.
4. Review any student concerns flagged in class logs.
5. Update set_protocol.md with any score data received.
